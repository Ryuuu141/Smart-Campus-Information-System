## About

The Smart Campus Information System is a command-line Python application that digitally manages student information, academic records, and performance analysis. It replaces manual administrative tasks such as student registration, course enrollment, record maintenance, searching, fee calculation, and performance analytics.

Each module corresponds to a lab experiment (Q1–Q8), and together they form the integrated Q9 Mini Project.

## Project Structure

```
smart_campus_is/
│
├── main.py                  # Entry point — main menu dashboard (Q9)
│
├── modules/
│   ├── __init__.py
│   ├── registration.py      # Q1 — Student registration & grade evaluation
│   ├── enrollment.py        # Q2 — Course enrollment management
│   ├── records.py           # Q3 — Student record data management
│   ├── search_sort.py       # Q4 — Sorting and searching student IDs
│   ├── fees.py              # Q5 — Fee calculation using functions
│   ├── file_handling.py     # Q6 — File handling for academic records
│   ├── directory.py         # Q7 — Directory scanning & exception handling
│   └── analytics.py         # Q8 — Performance analytics (NumPy/Pandas)
│
├── data/
│   └── student_records.csv  # Auto-generated by Q6 file handling module
│
├── charts/                  # Auto-generated by Q8 analytics module
│   ├── avg_scores.png
│   └── student_comparison.png
│
├── requirements.txt
└── README.md
```

## Modules

| # | Module | File | Key Concepts |
|---|--------|------|-------------|
| Q1 | Student Registration & Grade Evaluation | `modules/registration.py` | `if/elif/else`, functions, input validation |
| Q2 | Course Enrollment Management | `modules/enrollment.py` | `while` loop, `continue`, `break` |
| Q3 | Student Record Data Management | `modules/records.py` | Lists, dictionaries, sets (`&`, `\|`, `-`) |
| Q4 | Search & Sort Student IDs | `modules/search_sort.py` | Bubble sort, selection sort, linear & binary search |
| Q5 | Fee Calculation | `modules/fees.py` | Functions with default parameters |
| Q6 | File Handling | `modules/file_handling.py` | `open()`, `csv`, read/write/report |
| Q7 | Directory Scanning | `modules/directory.py` | `os.walk()`, custom exceptions |
| Q8 | Performance Analytics | `modules/analytics.py` | `numpy`, `pandas`, `matplotlib` |
| Q9 | Mini Project Integration | `main.py` | All of the above |

## Getting Started

### Prerequisites

- Python 3.8 or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/smart-campus-is.git
   cd smart-campus-is
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python main.py
   ```

Enter the number of the module you want to run and follow the prompts.
